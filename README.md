## üìã Features

* **Customizable Length:** Choose a password length from 6 to 24 characters using an intuitive slider.
* **Real-time Length Display:** The selected password length updates on the UI instantly as you move the slider.
* **Character Options:** Fine-tune your password by including or excluding:
    * Lowercase (a-z)
    * Uppercase (A-Z)
    * Numbers (0-9)
    * Symbols (!@#$...)
* **One-Click Copy:** Instantly copy the generated password to your clipboard with a single button click.
* **User Feedback:** The copy button provides visual feedback (changing to a "‚úÖ") upon successful copy.
* **Error Handling:** Displays a clear message ("Select at least 1 option!") if the user tries to generate a password without selecting any character types.
* **Responsive Design:** A clean, modern interface that looks and works great on both desktop and mobile devices.
